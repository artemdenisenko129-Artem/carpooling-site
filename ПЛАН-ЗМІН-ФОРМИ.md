# План змін форми оголошення та пошуку

## 1. Форма /new — що прибираємо

### "Хто я" — прибрати заголовок, залишити кнопки
- Зараз: `<p>Хто я</p>` + кнопки "Водій" / "Пасажир"
- Стане: просто кнопки "🚗 Водій" / "💺 Пасажир" без заголовку над ними

### Блок "Приміська/Міжміська" — видалити повністю
- Видалити кнопки "Приміська/Міська" і "Міжміська"
- Видалити тип `TripScope`, поле `tripScope` з форми
- Видалити валідацію `if (!form.tripScope)`
- В API — прибрати `tripScope` з destructuring і з об'єкту збереження
- В AnnouncementCard — прибрати чіп "🛣 Міжміська"
- На сторінці /announcement/[id] — прибрати чіп "🛣 Міжміська"

### Перейменування типу поїздки
- Зараз: заголовок "Як часто їдеш?" + кнопки "🔄 Регулярно" / "📅 Одного разу"
- Стане: маленький текст "поїздка" зверху + великі кнопки "🔄 Регулярна" / "📅 Разова"

---

## 2. Текст-підказка у формі (PLACEHOLDER_EXAMPLE)

### Зараз:
> Їду щодня з Ірпінь (ЖД вокзал) до Києва (Оболонь) і назад.
> Виїзд ~7:00, назад ~18:00. Через Гостомель, КПП.
> Є 1 місце. Пишіть в особисті.

### Стане (заголовок поля: "Розкажіть про умови поїздки"):
> Наприклад: місце зустрічі, вартість проїзду, тип та марка авто, наявність місця в багажнику — все, що хочете повідомити попутникам.

---

## 3. Поле "Спільнота" — додати пояснення

Під полем введення додати дрібний текст:
> ЖК, підприємство або університет — для пошуку поїздок з сусідами, колегами або однокурсниками.

---

## 4. Пошук (FeedPage) — що прибираємо

- Видалити кнопки "🏘 Приміська" / "🛣 Міжміська" з рядка фільтрів
- Видалити тип `ScopeFilter`, стан `scopeFilter`, логіку фільтрації по `tripScope`
- Кнопка "Всі" стає ресетом лише ролей (водій/пасажир) — як і зараз

---

## 5. Де ще зустрічається tripScope — прибрати

| Файл | Що прибрати |
|------|-------------|
| `app/new/page.tsx` | тип TripScope, поле форми, валідація, UI блок |
| `app/api/announcements/route.ts` | tripScope з destructuring і з doc |
| `components/FeedPage.tsx` | ScopeFilter тип, scopeFilter стан, фільтрація, кнопки |
| `components/AnnouncementCard.tsx` | чіп "Міжміська" |
| `app/announcement/[id]/page.tsx` | чіп "Міжміська" |
| `types/announcement.ts` | поле `tripScope` (залишити для сумісності зі старими записами — не обов'язково) |

---

## Питання перед виконанням

1. Кнопку "Всі" в пошуку — залишити (скидає роль на "всі") чи прибрати теж?
2. У старих оголошеннях в БД є `tripScope` — нічого страшного, поле просто більше не використовується.
