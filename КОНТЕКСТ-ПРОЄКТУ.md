# Контекст проєкту «ПопуткиUA»

Цей файл — головний орієнтир. Передай його Claude на початку нової сесії.

---

## Що це

Веб-сервіс пошуку попутників для щоденних приміських поїздок.
Водії та пасажири публікують оголошення, знаходять один одного через Telegram.

**Сайт:** https://carpooling-site.vercel.app  
**Канал:** https://t.me/poputky_ua  
**Бот:** @Poputtky_bot  

---

## Технічний стек

| Шар | Технологія |
|-----|-----------|
| Фреймворк | Next.js 15, App Router, React 19, TypeScript |
| Стилі | Tailwind CSS + власні токени в globals.css |
| БД | MongoDB Atlas, база `carpooling`, колекція `announcements` |

| Аналітика | GA4 (NEXT_PUBLIC_GA_ID=G-7R9FDQHSGJ) + власний лічильник views у MongoDB |

## Ключова бізнес-логіка

**Cron** (щодня о 02:00 UTC, vercel.json):
- Деактивує ТІЛЬКИ `tripType: "once"` з `departureDate < today`
- Регулярні (`tripType: "regular"`) — ніколи не чіпає

**Адмін** (artemdenisenko129@gmail.com):
- Статистика, список оголошень з пагінацією
- Деактивація / відновлення / bulk-деактивація (soft delete)
- Сортування: за датою (asc/desc), за переглядами

**Telegram-канал** (@poputky_ua):
- При кожному новому оголошенні бот (@Poputtky_bot) автоматично постить у канал
- Повідомлення: роль, маршрут, розклад, час, контакт
- Посилання НЕ включено (щоб уникнути превʼю з рекламою)
- Бот — адміністратор каналу з правом публікації

**Маркери на карті:**
- Неактивний стан: сірий колір, жовте кільце (#FFE500)
- Активний (при кліці): колір ролі (водій=#E24B4A, пасажир=#378ADD), біле кільце
- Без кластеризації — кожен маркер окремо
- `highlightId` prop: при переході з картки списку → карта автоматично відкриває маршрут

## Важливі технічні деталі

**Leaflet + React 19:**
Leaflet завантажується через CDN `<Script strategy="beforeInteractive">` в layout.tsx.
У LeafletMap.tsx і MapPicker.tsx — тільки `window.L` синхронно.
Ніяких `import("leaflet")` — React 19 hydration remount їх вбиває.

**CIFS mount (Cowork):**
Файли > ~4-5kb обрізаються при записі через Edit/Write.
Для великих файлів використовувати Python:
```python
python3 << 'EOF'
with open('/sessions/.../file.tsx', 'w', encoding='utf-8') as f:
    f.write(content)
EOF
```

**PowerShell термінал:**
Не підтримує `&&`. Команди запускати по одній:
```
git add -A
git commit -m "..."
git push
```
