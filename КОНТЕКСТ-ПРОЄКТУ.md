## Пагінація (реалізовано 2026-05-31)

- Перший рендер: **20 оголошень** (server-side, було 40)
- Кнопка "**Показати ще 20**" внизу списку (client fetch)
- API: `GET /api/announcements?page=0&limit=20&from=...&to=...` → `{ items, hasMore }`
- Пошук (from/to) шукає по **ВСІЙ базі** MongoDB (не тільки по завантаженим)
- Фільтри (роль/тип/спільнота) — по завантаженим (нормально для UX)
- При зміні пошуку — `key` prop скидає стан FeedPage

## Бізнес-логіка

**Cron** (щодня 02:00 UTC): деактивує `tripType:"once"` з `departureDate < today`.  
Регулярні — ніколи не чіпає.

**Антиспам:** не більше 5 активних оголошень + не частіше 1 на 10 хвилин.

**Telegram-канал** (@poputky_ua): при кожному новому оголошенні бот постить автоматично. БЕЗ посилання на сайт.

## Z-index (критично для карти!)

- Map wrapper: `position:relative; zIndex:1` — ізолює Leaflet panes
- Sticky header: `z-index:50`
- PlaceAutocomplete dropdown: `zIndex:9999`
- **Причина:** Leaflet panes мають z-index 200-700 у root context без ізоляції
