## Бізнес-логіка

**Cron** (щодня 02:00 UTC): деактивує `tripType:"once"` з `departureDate < today`.
**Антиспам:** max 5 активних + не частіше 1 на 10 хв.
**Telegram:** кожне нове оголошення → автопост у @poputky_ua. Без посилання на сайт.

## Критичні технічні деталі

**Leaflet:** тільки `window.L` — ніяких `import("leaflet")` або `next/dynamic` для Leaflet.
**Z-index:** map wrapper `zIndex:1` → ізолює panes. Header `z-50`. Autocomplete `z-9999`.
**Кольори:** водій=червоний (#E53935), пасажир=синій (#3A6BBF) — скрізь однаково.

**CIFS truncation (важливо!):** файли >4-5kb обрізаються через Edit/Write.
Використовувати Python для великих змін:
```bash
python3 -c "
with open('file.tsx','rb') as f: c=f.read().decode()
c = c.replace('old','new')
with open('file.tsx','wb') as f: f.write(c.encode())
"
```
Перевірка після змін: `npx tsc --noEmit` — нуль помилок = все ок.
